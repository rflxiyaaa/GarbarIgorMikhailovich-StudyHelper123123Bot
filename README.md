# GarbarIgorMikhailovich-StudyHelper123123Bot

Telegram-бот **«Помощник учёбы»** (StudyHelperBot).

**Автор:** Гарбар Игорь Михайлович (Garbar Igor Mikhailovich)

## Описание

Бот с кнопками-ссылками для учёбы: StackOverflow, мануал по Telegram-ботам, документация C#, GitHub, страница колледжа.

## Запуск

1. Укажите токен бота в **Properties → Resources.resx** (ресурс `Token`).
2. Запустите проект (F5).

## Технологии

- C# / .NET 8
- Telegram.Bot
